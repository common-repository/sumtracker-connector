Plugin to connect your woocommerce store with sumtracker.

== Description ==

Sumtracker connector is used to connect your woo commerce store with sumtracker inventory management software.

Steps to use the plugin:
1. Install woo commerce if you don’t have it already
2. Install the sumtracker connector plugin
3. Using the button displayed in the plugin you can login to sumtracker
4. In sumtracker, you have to click the connect store button
5. Select woo commerce and enter your store url. Then click on connect button
6. This will take you to the permission page for woo commerce oauth. On completing the oauth process you will come back to sumtracker

The above steps will connect sumtracker software with your woo commerce store. Now you can start managing your inventory using sumtracker. Read more about sumtracker here: https://help.sumtracker.com

Once you have connected your woo commerce store with sumtracker the plugin will show “connected” status.

Learn more here: https://sumtracker.com
